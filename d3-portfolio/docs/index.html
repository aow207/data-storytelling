D3 Portfolio

We're remaking static graphics into D3.js

Andrew Williams

Published October 11, 2019

Build a portfolio website using sawhorse. This website should include:

  • Two of your projects from this summer converted into D3
  • Your "best" graphic redone as ai2html (you did this already, see Homework 01, you just need to insert it into your sawhorse template)

Host this as a live website using GitHub Pages.

It's OK for redo your ai2html project as a D3 project, just make sure it's up there both as an ai2html project and as a D3 project.

Submit the URL to your live website (not the GitHub repo).

Each D3 graphic needs:

  • Title and subhead (although this can be in the text instead of in the graphic)
  • To look awesome
  • At least one annotation or tooltips
